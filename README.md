# TripLite · 出境轻导游

> 澳门 → 芭提雅 → 曼谷 出境旅行引导助手
> 行程：2026年4月26日 – 5月3日（8天）
> 定位：工具型旅行助手，弱网/离线可用，支持 PWA 安装到桌面

## 🏷️ v1.0 · 纯静态版（当前版本）

> 发布时间：2026年4月
> 技术栈：纯 HTML + CSS + JavaScript，无后端，无数据库
> 特点：**首次联网后完全离线可用**

### 已完成功能

#### 📱 PWA 应用化
- Service Worker 离线缓存（v7），首次联网后断网可用
- 支持安装到手机桌面（iOS Safari / Android Chrome）
- 智能安装引导弹窗，自动识别设备系统，分别展示操作步骤
- manifest.json 配置完整（图标、主题色、独立窗口模式）

#### 🏠 首页（index.html）
- 今日行程主卡（Day N / 城市 / 概述 / 体力标签）
- 整趟行程进度条 + 四城市里程碑点
- 今日要点（必做活动列表）
- 今日提醒（紧急/警告事项）
- 今日天气速览卡（实时/缓存/气候参考智能切换）
- 快捷入口（6个功能按钮）
- 全程日历时间轴（8天，点击跳转详情）
- TDAC 填写提醒（含官网直达链接）

#### 🗓️ 行程页（trip.html）
- 8天完整行程，每天三时段（上午/下午/晚上）
- 活动分级：必做 / 建议 / 可选
- 城市标签筛选（澳门/芭提雅/曼谷）
- 每日体力值、警告提醒、备选方案
- 相邻日期切换（上一天/下一天）
- URL参数直跳：`trip.html?day=d1`

#### 🗺️ 地图页（map.html）
- Leaflet 交互地图（OpenStreetMap底图）
- 全程 POI 标记点（7种类型：景点/住宿/交通/美食/夜生活/购物/按摩）
- 城市切换（全部/澳门/芭提雅/曼谷）
- 每个地点双导航按钮：📍高德地图（国内/澳门用）+ 🗺️谷歌地图（泰国用）
- 复制地址功能
- 路线卡片（6条行程路线说明）
- POI 类型彩色图例

#### ✅ 清单页（checklist.html）
- 6大分组（证件/入境/网络/支付/行李/App）
- 单项/全组勾选，localStorage 持久化
- 出发倒计时横幅（颜色随天数变化）
- 阶段筛选（出发前/出发当天/途中）
- 完成进度统计 + 未完成项高亮
- 5大关键事项 Top 提示块

#### ☀️ 天气页（weather.html）
- **三级智能数据源**：
  - 有网 + 临近出发10天内 → 和风天气实时预报
  - 断网有缓存 → 3小时内缓存预报
  - 无预报/无网 → 气候参考数据（兜底）
- 数据来源标签（🟢实时/🟡缓存/🔵气候参考）+ 手动刷新按钮
- 8天逐日天气卡（可展开：UV进度条/体感/穿搭建议/当日必做）
- 全程核心风险速查（高温/紫外线/脱水/雷阵雨/海浪）
- 防暑防晒指南（9大类，可折叠）
- 城市气候对比表（澳门/芭提雅/曼谷）
- 行程阶段智能横幅（距出发X天/旅途中Day N/已结束）

#### 🏨 住宿页（stay.html）
- 三城市（澳门1晚/芭提雅2晚/曼谷3晚）行程概览卡
- 9家酒店推荐（含价格/评分/标签/优缺点）
- 三平台预订按钮：Booking + Agoda + Trip.com（国内可用）
- 各城市住宿注意事项（退房时间/机场接送等）

#### 🛡️ 应急页（emergency.html）
- **纯 HTML 静态内容，无JS依赖，弱网优先**
- 紧急电话（点击直拨：使馆/旅游警察/急救/警察）
- 网络方案：eSIM推荐（Airalo/Nomad）+ 实体SIM + 漫游备用 + 网络失联应急
- 出行指南：Grab使用步骤 + Bolt备用 + 正规出租车识别
- 支付指南：现金/刷卡场景对比表 + 换汇技巧 + 银行卡注意事项
- 突发状况：护照丢失流程 + 就医指南 + 常见骗局
- 应急泰语短句（点击复制）
- 出发前必须备份信息清单

```
TripLite/

├── stay.html           住宿页
├── emergency.html      应急页
├── manifest.json       PWA配置
├── sw.js               Service Worker（离线缓存 v7）
├── css/
│   └── style.css       全局样式
├── js/
│   ├── data.js         核心数据（行程/地图/清单/住宿/应急/天气）
│   ├── home.js         首页渲染逻辑
│   ├── weather-service.js  和风天气API服务模块
│   └── app.js          公共逻辑（导航/SW注册/PWA安装引导）
└── icons/
    ├── icon-192.png    PWA图标
    └── icon-512.png    PWA图标
```

### 数据对象说明（js/data.js）

| 对象 | 说明 | 主要字段 |
|------|------|---------|
| `TRIP_DATA` | 8天行程 | days[].times[].activities[]（type: must/suggest/optional）|
| `MAP_DATA` | 地图数据 | pois[]（7种type）+ routes[] |
| `CHECKLIST_DATA` | 行前清单 | 6组，每项含 phase/required 标记 |
| `STAY_DATA` | 住宿推荐 | 3城市，9家酒店，含booking/agoda/trip链接 |
| `EMERGENCY_DATA` | 应急信息 | 机场/延误/行李/断网处理 |
| `WEATHER_DATA` | 天气基础数据 | 8天气候参考 + topRisks + heatGuide + cityComparison |

### 外部依赖

| 依赖 | 用途 | 离线降级方式 |
|------|------|------------|
| FontAwesome 6.4 CDN | 图标 | 降级显示emoji，不影响内容 |
| Google Fonts (Inter) | 字体 | 降级系统字体 |
| Leaflet 1.9 CDN | 地图底图 | 地图背景空白，POI标记仍可见 |
| 和风天气 API | 天气预报 | 降级气候参考数据 |

### 功能入口

```
/index.html              首页（今日概览）
/trip.html               行程总览
/trip.html?day=d1        直跳第1天（d1–d8）
/map.html                地图（默认全部城市）
/checklist.html          行前清单
/weather.html            天气预报 & 注意事项
/stay.html               住宿推荐
/emergency.html          应急 & 实用指南
```

### 已知限制

- 地图底图需联网（Leaflet + OpenStreetMap），离线时地图背景为空白
- 天气预报在出发10天前不可用（超出和风天气免费版预报范围）
- 清单勾选状态仅存本机（localStorage），换设备需重新勾选
- 所有行程/地图数据写死在 data.js，更新需重新发布

## 🔮 v2.0 · 后端版（计划中）

> 新增服务器 + 数据库，实现数据持久化和多人协作
> 技术栈：Node.js + Express + PostgreSQL

### 计划功能

#### 📓 旅行日记
- 每天可写文字游记，带时间/地点标签
- 手绘涂鸦（Canvas画板 → 存图片）
- 上传手机照片（存阿里云OSS）
- 数据结构：diary { day, city, text, images[], drawings[], created_at }

#### 👥 多人行程共享
- 创建"旅行团"，生成邀请码
- 同行人加入后共享同一份行程
- 行程修改实时同步

#### 💰 费用分摊记录
- 记录每笔消费（谁付/金额/类别）
- 自动计算AA账单
- 行程结束生成费用报告

#### ☁️ 清单云同步
- 勾选状态存服务器
- 多设备实时同步
